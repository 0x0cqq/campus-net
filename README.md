# campus-net

FAKE THURSDAY UNIVERSITY NETWORK

## Project setup

```
npm install
pip install -r requirements.txt
```

## Develop

### Lints and fixes files

```
npm run lint
```

## Deploy

### Local Deploy

1. Compiles Static files(frontend)

```
npm run build
```

2. Setup Flask server(backend)

```
python3 main.py
```
